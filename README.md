# Charts

Scalable Data Charts created with SVG graphics.

#### [Check it out on github.io](https://michudud.github.io/Charts/ "Charts-github.io")

## Features

- Bar Chart
- Line Chart
  - markers on/off

### How to open

- Install: `npm install`
- Run: `npm start`

### Stack

- JavaScript
- React
- HTML
- Styled Components
